# Python Mini Projects Collection

## Overview

This repository contains a collection of Python mini projects developed for learning programming fundamentals, problem-solving, file handling, encryption, game development, and graphical applications.

These projects were created to strengthen Python programming skills through practical implementation of real-world concepts.

---

## Projects Included

### 1. Quiz Game

A command-line quiz application that asks computer-related questions and calculates the user's score.

**Concepts Used**

* Variables
* Conditional Statements
* User Input
* String Handling

---

### 2. Number Guesser

A game where the user attempts to guess a randomly generated number within a specified range.

**Concepts Used**

* Loops
* Random Module
* Input Validation

---

### 3. Rock Paper Scissors

A classic game where the user competes against the computer.

**Concepts Used**

* Random Module
* Loops
* Conditional Logic

---

### 4. Mad Lib Generator

A story generator that replaces placeholders with user-provided words.

**Concepts Used**

* File Handling
* Dictionaries
* String Manipulation

---

### 5. Choose Your Own Adventure

An interactive story game where the user's decisions determine the outcome.

**Concepts Used**

* Conditional Statements
* Branching Logic
* User Interaction

---

### 6. Times Math Challenge

A math quiz that generates random arithmetic problems and measures completion time.

**Concepts Used**

* Time Module
* Random Module
* Arithmetic Operations

---

### 7. Pig Dice Game

A multiplayer dice game where players compete to reach the target score.

**Concepts Used**

* Functions
* Lists
* Loops
* Game Logic

---

### 8. Slot Machine

A console-based slot machine simulation with betting and winnings calculation.

**Concepts Used**

* Random Module
* Functions
* Dictionaries
* Game Development

---

### 9. Password Manager

A secure password manager that encrypts and stores passwords using Fernet encryption.

**Concepts Used**

* Cryptography Library
* File Handling
* Encryption and Decryption

---

### 10. Turtle Racing Game

A graphical turtle racing simulation using Python's Turtle module.

**Concepts Used**

* Turtle Graphics
* Object-Oriented Concepts
* Randomization
* GUI Programming
